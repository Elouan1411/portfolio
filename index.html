Application de Completion semi-automatique

Application de complétion semi-automatique développée en Rust et Python. Elle propose des suggestions de mots intelligentes pour l'utilisateur en s’appuyant différentes approches.

  • Rust
  • Python
  • Linux
Code source

La première étape de ce projet a consisté à réaliser un travail de recherche approfondi sur les techniques utilisées dans la complétion semi-automatique de texte. Ce travail, mené en collaboration avec ma camarade Samia Benali, s’inscrivait dans le cadre du Cursus Master Ingénierie (CMI) durant ma deuxième année de licence (L2).

Nous avons étudié différentes approches utilisées par les systèmes modernes, notamment les algorithmes de distance d’édition tels que celui de Levenshtein, ainsi que l’usage des chaînes de Markov dans l’adaptation des suggestions en fonction du contexte ou de l’historique de l’utilisateur.

À la suite de cette phase d’analyse, j’ai conçu et développé une application de complétion semi-automatique permettant de mettre en œuvre ces concepts de manière concrète. L’objectif est d’offrir des suggestions pertinentes à partir de mots partiellement saisis.

L’application repose sur trois approches principales :

  • La distance de Levenshtein pour mesurer la similarité entre les mots et corriger les fautes de frappe ;
  • La reconnaissance de préfixes pour proposer rapidement des complétions adaptées aux premières lettres tapées ;
  • La fréquence d’utilisation pour favoriser les mots les plus couramment employés par l’utilisateur.

Le cœur du projet est développé en Rust, garantissant performance et sécurité mémoire, tandis que Python gère la partie interface et automatisation. L’installation et l’exécution sont simplifiées grâce à une commande unique make install.

Ce projet m’a permis de combiner travail de recherche et développement logiciel, tout en approfondissant mes connaissances en traitement du langage naturel, optimisation algorithmique et programmation système.

Canyon Radar 🧗‍♂️

Application web pour planifier vos sorties canyoning. Choisissez un lieu ou utilisez votre position actuelle, filtrez les canyons accessibles selon le temps de route et exportez les résultats pour planification.

  • HTML
  • CSS
  • JavaScript
  • Node.js
Code source

Canyon Radar permet de trouver tous les canyons accessibles depuis un point donné en un temps de route maximal. Les résultats peuvent être exportés en CSV ou Excel pour organiser vos sorties.

Voir le site

Le projet utilise l’API OpenRouteService pour calculer les temps de trajet précis et un backend Node.js hébergé sur Vercel pour sécuriser la clé API.

Les données des canyons sont stockées dans data.json et filtrées côté frontend pour afficher uniquement les lieux accessibles. Le frontend est développé en HTML, CSS et JavaScript.

Canyon Radar intègre plusieurs fonctionnalités clés :

  • Choix de la localisation : position actuelle ou point sur la carte ;
  • Filtrage par temps de trajet : limite les canyons accessibles ;
  • Exportation des résultats : CSV ou Excel pour planification et analyse.

Ce projet m’a permis d’explorer l’intégration d’API tierces, la gestion de données géographiques et le développement d’une interface web interactive.

Centipede Game 🐛

Centipede Game est un projet réalisé dans le cadre du projet semestriel de L1 en CMI Informatique. Il s’agit de mon premier projet de développement de jeu vidéo, entièrement conçu en Python à l’aide de Pygame.

  • Python
  • Pygame
Code source

Le joueur incarne un nain et doit éliminer des vagues successives d’ennemis variés. À chaque nouveau niveau, la vitesse et le nombre d’ennemis augmentent, rendant le jeu de plus en plus difficile.

Le jeu propose un système de niveaux progressif et une sauvegarde du meilleur score après chaque partie. La programmation a été réalisée en POO (Programmation Orientée Objet) avec l’utilisation de l’héritage pour structurer les entités du jeu (joueur, ennemis, projectiles...).

Ce projet m’a permis de découvrir le développement de jeux 2D, la gestion d’événements clavier, les collisions, et les boucles principales de jeu.

Reconnaissances :

  • Samia Benali pour la création des images utilisées dans le jeu ;
  • Nos professeurs et mentors pour leur accompagnement pendant le projet.

Un rapport de projet détaillant la conception, la structure du code et les choix techniques a également été rédigé dans le cadre de cette réalisation.

📌 R&D Day - Promo 2025

Ce site présente la Journée Recherche & Développement organisée par notre promotion du CMI Informatique. L’édition met en avant les avancées en informatique quantique et leur lien avec la physique, sous le thème :

🎭 Physique et informatique quantique : le duo qui redéfinit l’avenir

  • HTML
  • CSS
  • JavaScript
  • PHP
Voir le site Code source

L’événement met en lumière les travaux de recherche et d’innovation des étudiants à travers des stands interactifs, des présentations scientifiques et des activités ouvertes à tous.

Le site a été refondu en binôme avec un camarade afin de moderniser son interface et d’améliorer l’expérience utilisateur. Nous avons également adapté l’identité visuelle et le contenu au thème de la journée que nous avions choisi : « Physique et informatique quantique : le duo qui redéfinit l’avenir ».

Cette refonte avait pour objectif de rendre la navigation plus fluide et de permettre aux visiteurs de se projeter pleinement dans l’événement grâce à une présentation claire, immersive et cohérente avec l’esprit de la journée.

Le site offre ainsi une navigation responsive, permettant d’accéder facilement au programme de la journée, aux informations pratiques (lieu, horaires, organisation), ainsi qu’à la présentation du thème principal.

Fonctionnalités principales :

  • Présentation du thème et du programme de la journée ;
  • Informations pratiques sur l’organisation de l’événement ;
  • Mise en avant des stands et activités ;
  • Design responsive pour une consultation sur mobile et PC.

Déploiement : le site est hébergé sur ofni.asso.fr.

Remerciements : ce site repose sur une base développée par nos prédécesseurs de la promotion précédente, que nous avons adaptée et améliorée pour cette édition. Un grand merci à eux pour leur travail et leur transmission ! 🎓

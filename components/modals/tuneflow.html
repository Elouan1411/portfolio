TuneFlow - Découverte Musicale 🎵

TuneFlow est une application Android native développée de zéro en 7 jours. Ce projet était un défi pour apprendre Kotlin et l'écosystème Android de manière intensive.

  • Kotlin
  • Android SDK
  • API
Code source Télécharger l'APK

L'objectif de ce projet était de maîtriser la syntaxe de Kotlin et les principes fondamentaux du développement Android (Activités, Fragments, Cycles de vie, RecyclerView).

TuneFlow permet aux utilisateurs de :

  • Découvrir de nouvelles musiques basées sur leurs goûts (algorithme de recommandation) ;
  • Créer et gérer des playlists ;
  • Partager des playlists avec des amis ;
  • Explorer des musiques par humeur ;
  • Profiter d'un mode clair et sombre.

Plus d'informations sont disponibles sur le dépôt GitHub, y compris le code source complet et la version téléchargeable.
